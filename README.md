# AntiIllegalRecode

**AntiIllegalRecode** is a fully **re-coded** version of the classic AntiIllegal plugin for Paper 1.21.4.  
It protects your server by detecting and removing illegal items, blocks, and overleveled enchantments.

---

## Features

- Removes unobtainable blocks (Bedrock, Barrier, Command Blocks, etc.)  
- Prevents overstacked items and invalid amounts  
- Clamps or removes illegal enchantments  
- Notifies players when illegal items are removed  
- Toggle the plugin on/off with `/airecode <enable|disable>`  

---

## Installation

1. Place `AntiIllegalRecode.jar` in your server `plugins` folder  
2. Start or restart your Paper server  
3. The plugin generates a default `config.yml` automatically
